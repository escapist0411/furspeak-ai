# 🐶 FurSpeak AI

## 🚀 Features
- Dog & Cat emotion detection 🐶🐱
- MFCC feature extraction
- SVM model
- Analytics dashboard
- History tracking

- Librosa

- MySQL
- HTML/CSS/JS

- Angry 😡
- Sad 😢
