🐶 FurSpeak AI

Understanding animal emotions through Artificial Intelligence 🧠

📌 Project Overview

FurSpeak is an AI-powered emotion detection system designed to analyze animal sounds and interpret their emotional state. The system processes audio signals and classifies them into emotions such as Happy, Angry, Sad, and Hungry.

⚙️ How It Works

  • Audio input is captured or uploaded by the user
  • MFCC features are extracted using Librosa
  • A trained Machine Learning model (SVM) processes the features
  • The system predicts the emotional state
